# Sistem Pendaftaran Klinik

Repositori ini adalah root untuk proyek Sistem Pendaftaran Klinik. Proyek ini dibagi menjadi dua bagian utama:

1. [Frontend (fe)](./fe) - Aplikasi client menggunakan Vite + React.
2. [Backend (be)](./be) - Aplikasi server menggunakan Node.js + Express.

## Struktur Folder

- `fe/`: Repositori Frontend
- `be/`: Repositori Backend
